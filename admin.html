🌍 Quản lý Translations

Tự động load từ Google Sheet > TRANSLATIONS. Sửa trực tiếp trong Sheet, website tự cập nhật!

📋 Cấu trúc TRANSLATIONS Sheet:

key vi en auto_translate
hero_title Khám phá Việt Nam Discover Vietnam true
nav_all Tất cả All true

💡 Cách sử dụng:

  1. Thêm key mới: Thêm row mới trong TRANSLATIONS sheet
  2. Sửa text: Edit trực tiếp cột vi/en trong Sheet
  3. Auto-translate: Để trống cột 'en', set auto_translate=TRUE → Tự động dịch
  4. Refresh: Website tự load translations mỗi lần refresh (cache 5 phút)
