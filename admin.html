✍️ Tạo Post Thủ Công

Tạo bài viết mới bằng cách điền form bên dưới. Content phải là HTML.

Chỉ dùng chữ thường, số, và dấu gạch ngang. VD: top-10-da-nang
150-200 ký tự. Hiển thị trong post card và search results.
Chọn text → Click button để format
💡 Cách viết đơn giản:
  • Viết bình thường như trong Word
  • Mỗi đoạn cách nhau 1 dòng trống
  • Dòng bắt đầu bằng số (1., 2.) = tiêu đề section
  • Dòng ngắn (<50 ký tự) = tiêu đề
  • Dòng có dấu ":" ở cuối = bullet point
  • Hệ thống tự động chuyển sang HTML! ✨
Sử dụng HTML tags: <h2>, <p>, <ul>, <li>, <strong> 📚 Xem hướng dẫn HTML

📚 HTML Quick Reference

Headings (Tiêu đề):

<h2>Section Title</h2>
<h3>Subsection Title</h3>

Paragraphs (Đoạn văn):

<p>This is a paragraph.</p>

Bold Text (In đậm):

<strong>Important text</strong>

Lists (Danh sách):

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

Links (Liên kết):

<a href="https://example.com">Click here</a>

🌍 Quản lý Translations

Tự động load từ Google Sheet > TRANSLATIONS. Sửa trực tiếp trong Sheet, website tự cập nhật!

📋 Cấu trúc TRANSLATIONS Sheet:

key vi en auto_translate
hero_title Khám phá Việt Nam Discover Vietnam true
nav_all Tất cả All true

💡 Cách sử dụng:

  1. Thêm key mới: Thêm row mới trong TRANSLATIONS sheet
  2. Sửa text: Edit trực tiếp cột vi/en trong Sheet
  3. Auto-translate: Để trống cột 'en', set auto_translate=TRUE → Tự động dịch
  4. Refresh: Website tự load translations mỗi lần refresh (cache 5 phút)
